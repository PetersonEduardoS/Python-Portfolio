## Project 01 - Simple Calculator

This is a simple command-line calculator built with Python.

### Features

- Addition
- Subtraction
- Multiplication
- Division
- Division by zero validation
- Invalid operation validation

## Project 02 - EUR to BRL Converter

This is a simple currency converter that converts an amount in euros to Brazilian reais using a manually entered exchange rate.

### Features

- User input for euro amount
- User input for exchange rate
- Currency conversion calculation
- Output formatted with two decimal places

## What I learned

- How to use `print()`
- How to get user input with `input()`
- How to convert text input to numbers using `float()`
- How to use conditional statements with `if`, `elif`, and `else`
- How to format decimal numbers using f-strings
- How to perform basic arithmetic calculations
